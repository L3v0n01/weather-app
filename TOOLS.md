# Tools Used

## Claude Code (AI Assistant)

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) by Anthropic was
used as an AI-powered coding assistant throughout the project. It helped
accelerate development across a 14-module codebase — primarily for code
generation of repetitive patterns, tests, project-wide refactoring, and
systematic audits (hardcoded strings, magic numbers, missing content
descriptions, unused imports and dependencies). It also helped with planning
and structuring the work: breaking down the implementation into phases.
Documentation — including this file, the README, LICENSE, and commit
messages — was drafted with AI help and reviewed before use.

All generated code and documentation was reviewed and adjusted before
committing.

- **GitHub Actions** — CI pipeline with lint, test, coverage, and build
